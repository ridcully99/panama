Zeichnen Sie mit dem Finger auf dem Bildschirm.

Strichstärke ändern: Tippen Sie auf den Pinsel-Button und verwenden Sie den Schieberegler um die gewünschte Stärke einzustellen. Alternativ dazu können Sie die Pinselstärke auch mittels der Lautstärkenregler ändern.

Strich-Transparenz ändern: Tippen Sie auf den Transparenz-Button und verwenden Sie den Schieberegler um die gewünschte Transparenz einzustellen.

Farbe auswählen: Tippen Sie auf den Palette-Button um die Farbpalette anzuzeigen. Bewegen Sie den Finger auf Farbpalette um die gewünschte Farbe auszuwählen. Sie können auch eine Farbe aus Ihrem Bild auswählen, wenn Sie statt der Palette das Bild berühren (die Palette wird dann sofort ausgeblendet) und dann den Finger auf dem Bild herumbewegen bis Sie die gewünschte Farbe haben.

Striche rückgängig machen: Mittels Zurück-Button oder der entsprechenden Menü-Funktion

Menü Optionen

Neu: Bildschirm löschen und neues Bild beginnen. Sie können einen einfarbigen Hintergrund oder ein bestehendes Bild wählen um darauf zu zeichnen, oder Sie können mit der Kamera ein Bild aufnehmen und dieses als Hintergrund verwenden.

Speichern: Bild speichern im Pictures-Ordner Ihrer SD-card. Die Bilder sind benannt als fingercolors und dem Datum und Zeitpunkt an dem das Bild gespeichert wurde, z.B. fingercolors-110218-203155

Freigeben: Damit können Sie Ihr Bild auf Facebook, Picasa, Gmail, etc. veröffentlichen.

Rückgängig: Rückgängig machen von Strichen (uneingeschränkt).

Wiederherstellen: Wiederherstellen zuvor rückgängig gemachter Striche.
